Research focus

The direction I want to study during a PhD.

Self-evolving and open-ended agents with long-term memory

I want to study self-evolving and open-ended LLM-based agents: agents that do not only complete tasks, but also store experiences, retrieve past context, update strategies, build tools, and adapt their behavior over time. My concrete angle is long-term memory: how such agents should remember, revise, forget, and expose what they know when working over documents, code, scientific data, and knowledge bases.

Concrete question: How can long-term memory keep self-evolving and open-ended agents reliable, inspectable, private, and controllable as their behavior changes over time?

Built a predictive modelling pipeline for incomplete count time series using MICE and XGBoost with Poisson loss. The work combined scientific data preparation, time-aligned feature engineering, leakage-aware validation, and reproducible HPC experiments.

Independent project concept for a local-first AI memory layer using LLMs, embeddings, RAG, vector search, and possibly a knowledge graph to map files, folders, documents, and user context for retrieval and reasoning.
